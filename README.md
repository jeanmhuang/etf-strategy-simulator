# 📈 ETF Strategy Simulator

Analyze risk, return, and drawdowns for major ETFs using Python.

- 🧪 **Colab notebook**: Risk/return analysis with yfinance + plotly  
- 🎛️ **Streamlit app**: Interactive dashboard for ETF comparison
